# 📚 论文审核快速指南

## 如何审核每日论文？

当你收到 **"📚 Paper Review"** issue 通知后：

### 步骤 1: 打开 Issue

点击通知链接，或访问：
```
https://github.com/windrise/windrise.github.io/issues
```

### 步骤 2: 审核论文

浏览每篇论文的：
- 📊 **评分**（满分10分）
- 📝 **AI 摘要**
- 🔑 **关键贡献**
- 🔗 **arXiv 链接**

### 步骤 3: 添加标签 ⭐

**在 Issue 页面右侧找到 "Labels" 区域：**

1. 点击 **Labels** 旁边的齿轮图标 ⚙️
2. 选择相应的标签：

   - ✅ **`approved`** - 批准收录该论文
   - ❌ **`rejected`** - 拒绝该论文
   - ⭐ **`starred`** - 标记为重要论文

**就这么简单！**

### 步骤 4: 等待自动处理 🤖

一旦你添加了 `approved` 标签：

```
✅ 立即触发自动化流程（1-2分钟内）
    ↓
🤖 系统自动：
   1. 提取论文完整信息
   2. 自动分类到相应领域
   3. 添加到论文数据库
   4. 提交到 main 分支
   5. 触发网站重新部署
    ↓
💬 Bot 在 Issue 下评论确认
    ↓
🌐 5-10分钟后网站更新完成
```

你会收到：
- 📝 机器人在 Issue 中的确认评论
- 🔔 论文成功添加到网站的通知

### 步骤 5: 查看网站

访问你的网站，新论文已经出现在论文列表中了！

## 💡 使用技巧

### 批量审核

如果有多篇论文都想批准：
1. 对每篇都添加 `approved` 标签
2. 系统会一次性处理所有批准的论文

### 标记重要论文

同时添加 `approved` 和 `starred`：
```
1. 点击 Labels 齿轮图标
2. 同时勾选 approved 和 starred
3. 论文会被标记为"特别重要"
```

### 延后决定

不确定是否收录？
- 先不添加任何标签
- 点击论文链接深入阅读
- 随时回来添加标签

## 🔍 标签说明

| 标签 | 作用 | 自动化 |
|------|------|--------|
| `approved` | 批准添加到收藏 | ✅ 触发自动添加 |
| `rejected` | 拒绝该论文 | ❌ 不触发 |
| `starred` | 标记为重要 | 🌟 高优先级显示 |
| `paper-review` | 待审核（自动添加） | - |
| `automated` | 自动化任务（自动添加） | - |

## ❓ 常见问题

### Q: 添加 approved 标签后多久生效？

**A:** 通常 1-2 分钟内开始处理，5-10 分钟后网站更新完成。

### Q: 可以撤销吗？

**A:** 可以！移除 `approved` 标签即可。如果已经添加到数据库，需要手动编辑 `data/papers/papers.yaml` 删除。

### Q: 如果系统没反应怎么办？

**A:** 检查：
1. 是否添加了 `approved` 标签（不是评论"approved"）
2. 是否在 paper-review issue 上操作
3. 查看 GitHub Actions 是否运行中
4. 如果卡住，可以手动重新触发 workflow

### Q: 论文会自动分类吗？

**A:** 是的！系统根据标题和摘要自动分类到：
- 3D Gaussian Splatting
- 医学影像分析
- 神经辐射场 (NeRF)
- 3D 重建
- 自监督学习
- 等等

## 🎯 最佳实践

1. **每天花 5-10 分钟** 审核论文
2. **优先看评分高的**（≥ 8.0）
3. **检查是否有代码** - 有代码的论文更容易复现
4. **阅读 AI 摘要** - 快速了解核心贡献
5. **适度收录** - 保持高质量论文库
